# yaso

> yaso is a (de)compress tool writen in Python.


## Usage

```Python
$ yaso -c test test.zip
$ yaso -d test.zip -o test

```

## Build

`$ python3 main.py`

## Features

> Do not expect we will implement it all.

- Parameterization 
    - -c, compress
    - -d, decompress
    - ~~-S, customization~~ 
    - -o, ouput specific
- Format support
    - zip
    - rar
    - gz
    - tar.gz
- Algorithm support
    - compress and decompress algorithm(s)
    - crypt and decrypt algorithm(s)
